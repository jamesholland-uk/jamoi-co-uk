Description


The sentry robot is designed to track objects as they move around it. It does this by rotating its top tower as the target moves around, and also points a targeting arm at the target. The targeting arm changes angle in order to indicate that the distance between the target and the robot has been determined. LEDs and a laser diode illuminate the target when the target is in sight of the robot.


How it Works


The robot rotates back and forth until an object crosses the path of the middle infrared sensor on the robot. The distance between the target and the robot is then calculated using a predetermined formula, and then the target arm is set to point at the target. Then the LEDs and the laser switch on to indicate the target is in sight. When the object moves, the left and right infrared sensors detect which way the target moved, and instruct the tower to rotate in the appropriate direction until the target is found again.
